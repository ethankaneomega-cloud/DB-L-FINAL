# Stock Management System (SMS) - Final Project in DBSYS

## Introduction
The **Stock Management System (SMS)** is a web-based application developed to help businesses manage inventory efficiently. It handles tasks such as tracking items, managing suppliers, processing purchase orders, recording sales, handling returns, and monitoring stock levels in real-time. The system is designed to reduce manual errors, provide accurate inventory data, and streamline daily business operations.

## Features
- User authentication and role management
- Supplier management
- Item and stock tracking
- Purchase order creation and management
- Receiving of goods
- Sales transactions management
- Returns and back orders handling
- Dashboard monitoring and system settings

## Purpose
The purpose of SMS is to automate inventory management processes, ensure data accuracy, and provide businesses with a reliable system for decision-making based on real-time stock information.
